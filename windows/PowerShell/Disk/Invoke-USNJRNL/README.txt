1) Ensure you have the USNJRNL
2) In the directory where the USNJRNL file is, create anew directory and move the file into it
3) Open UsnJrnl2Csv.exe
4) Select the 'Browse $USNJRNL' button and navigate to the USNJRNL file
5) Click the 'Start Parsing' button
6) Once complete, open the .csv produced in Excel. Be sure to set the time column to "text". The times in the file are on UTC.
